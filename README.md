## 概要

データ構造とアルゴリズムに関して，Javaを用いてプログラムを作成し，そのプログラムが稼動することを確認する．プログラムは，毎週または隔週に一個の割合で作成する．


## 学習・教育目標

プログラミング技術の基本となるデータ構造とアルゴリズムを理解する．
基本的なデータ構造，アルゴリズムを用いたプログラミングの技術を習得する．
